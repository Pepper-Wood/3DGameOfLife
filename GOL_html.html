John 3D Game Of Life

This is the final graphics project for Kathryn DiPippo, Marina Espinoza, Darien Keyack, Alexander Mohr, and Kathrine Ryan.

Rules:

  • Cells (in this case, cubes) with only 1 or less neighbours die, as if by loneliness.
  • If 5 cells surround an empty cell, they breed and fill it.
  • If a cell has 8 or more neighbours, it dies from overcrowding.
